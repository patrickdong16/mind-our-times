# Mind Our Times 🔭

> AI-powered observatory of era-defining ideas and social consensus

每日追踪全球顶级思想者的观点，监测社会共识与时代情绪的微妙变化。

## 产品架构

| 层 | 职责 |
|---|---|
| **内容引擎** (Pepper) | 每日 RSS 扫描 → AI 分析 → 10 篇短分析 + 1 篇综述 |
| **数据层** (CloudBase) | 文章存储 + 投票数据 + 趋势统计 |
| **Webapp** | 今日/往期阅读 + 领域标签 + 评论 |
| **公众号** | 日度综述 + 时代问题投票 + 社交分发 |

## 六大观测领域

| 领域 | 核心问题 |
|------|----------|
| 🔧 技术 | AI 是否正在加剧社会分层？ |
| 🏛️ 政治 | 民主制度是否正在失效？ |
| 📜 历史 | 当前文明是否正在走向衰退？ |
| 💭 哲学 | 算法优化是否正在侵蚀自由意志？ |
| ⛪ 宗教 | 科技是否正在成为新宗教？ |
| 💰 金融 | 金融是否正在加剧社会撕裂？ |
